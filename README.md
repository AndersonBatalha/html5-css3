# Programação para internet com HTML5 e CSS3
## HTML5 
* Versão mais recente do HTML
* Estrutura básica

    ```
    <!DOCTYPE html>

    <html>
        <head>

        </head>

        <body>
        </body>
    </html>
    ```   

* Tags

    * Cabeçalho e rodapé
        ```
            <header>
                (...)
            </header>

            <footer>
                (...)
            </footer>
        ```

    * Parágrafo

        ```
        <p>Texto</p>
        ```

    * Título

        ```
        <h1>Texto</h1>
        <h2>Texto</h2>
        <h3>Texto</h3>
        <h4>Texto</h4>
        <h5>Texto</h5>
        <h6>Texto</h6>
        ```
